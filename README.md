📌 Project Overview

An end-to-end Data Analytics Project focused on analyzing customer shopping behavior using Python, SQL, PostgreSQL/MySQL/SQL Server, and Power BI.

This project uncovers valuable insights about:

Customer spending habits
Product performance
Subscription behavior
Discount impact
Revenue trends
Customer segmentation

The project follows the complete analytics lifecycle — from data cleaning and exploration to dashboard visualization and business reporting.

📂 Dataset Information

📊 Total Records: 3,900+ Customer Transactions
📑 Features Included:

Customer Demographics
Product Categories
Purchase Amount
Review Ratings
Shipping Methods
Discounts & Promo Usage
Subscription Status
Purchase Frequency
